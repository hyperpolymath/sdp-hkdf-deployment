= SDP HKDF Deployment

Rootless/SDP deployment for HKDF cryptographic services.

This repository provides the infrastructure for deploying HKDF (HMAC-based
Key Derivation Function) cryptographic services using the Secure Deployment
Protocol (SDP) with rootless container execution, air-gap readiness, and
provenance tracking.
